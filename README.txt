Objectives :-

	The objective of the project is to develop a bank managment software which will provide all the services needed by bank,
usually before the traditional way and they used to faced problems like data storage and retrival and the record were hand written and few computerized but this project aims at developing a command line driven bank management system which will be handled by trained professional.



	
Features :-

	To solve the problem of traditional banking system, Bank Management System has been developed. While
developing the systems, similar problems to traditional system were faced. The first was how to store and retrive
data efficiently. Secondly, how to create, update and delete accounts efficiently. Most important, to find a
solution to efficiently have a transaction without any errors and bugs.

Interface :-

	This project will provide a command line interface to the user of the bank. The user can,
	1) create acccount 
	2) update account 
	3) delete account
	4) balance inquiry

	Bank Management System has tried to solve all of the above problems in a efficient way.
